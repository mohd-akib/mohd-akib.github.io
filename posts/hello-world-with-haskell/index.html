Hello World With Haskell

January 31, 2020

I’ve mostly worked with imperative languages like (C/C++/Java/Python). This is the first time i’m learning any functional programming language (Haskell). It is really giving me new dimensions of reasoning about programs. It’s strongly and statically typed :-

Types of every function and variable are known at compile time.

Codes written in haskell are very much compact.

So without wasting time lets start by installing it.

curl -sSL https://get.haskellstack.org/ | sh

After this tools stack(tool to build haskell projects) and GHCi (glasgow haskell compiler interactive) will be available on your machine. You can start interative shell with command.

stack ghci

As soon as you hit stack ghci command interactive shell starts by loading prelude module. Prelude is a module that contains a small set of standard definitions and is included automatically into all Haskell modules.

Prelude>

That’s all folks , you’re ready to play with haskell.

Prelude> 2+2
4

to write multiple lines in the interactive shell you’ll have to use :{ in the beginning and :} at the end.

Prelude> :{
Prelude| 2+2
Prelude| :}
4

To define type of any variable or function :: keyword is used.

for example:-

x :: Integer

is read as x is a variable of type integer

fun :: Integer -> Integer

fun is a function that takes an integer and gives an integer as output.

fun :: Integer -> Integer -> Integer

fun is a function that takes two integer and gives an integer as output.

Ok. Let’s write Hello world in a file name hello.hs(hs is extension for haskell programs).


helloWorld :: IO()
helloWorld = putStrLn "Hello, World!"


helloWorld :: IO()

can be read as “hello is a function that does some IO operation”.

and the definition is

helloWorld = putStrLn "Hello, World!"

we can load any haskell code written in file using :load file_name

:load hello.hs

now our file is loaded in the environemnt. Now we can call any function written in the hello.hs file.

since it is having no arguments so we can call it with function name only, otherwise we’d have to pass the variables also.

*Main> helloWorld 
Hello, World!
