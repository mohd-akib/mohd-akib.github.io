Run Two Instance of Same Springboot Application in Intellij

November 4, 2019

Suppose i want to run two instances of my application. For this :

Run(top bar)-> Edit Configuration

Using left top corner + sign add configuration by choosing Application. Now you’ll have your existing application configuration and one “unnamed” configuration. Fill the “unnamed” configuration fields by copying from your application configuration of which you want to run another instance. Only two fields will differ - Name, VM options. In VM options in both the existing configuration and new configuration fill the field as -Dserver.port= (e.g. -Dserver.port=8001). Keep ports different in different configurations. To keep the configurations differentiable you can name them like :service+port

suppose my service name is SomeService and port for the configuration is set as 8001. Then you can name it as SomeService8001. And another configuration of same service setup to running on 8002 you can name it as SomeService8002. Naming like this will keep you able to differentiate multiple instances of same service.

Then Apply -> Ok

Then go to Run(in top bar)-> profile. There you can see all the configurations including your new one. Run from there one by one. And your two instances will be running on two different ports.
